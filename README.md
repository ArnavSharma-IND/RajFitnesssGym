# 🏋️ Raj Fitness Gym

A modern, premium, Apple-inspired fitness website designed for **Raj Fitness Gym**. Built using only **HTML, CSS, and JavaScript**, this project delivers a smooth, responsive, and visually engaging user experience with elegant animations and a luxury dark theme.

> **⚠️ Note for Reviewers:**  
> This project is complete and presentation-ready. **No changes are to be made.**

---

## ✨ Features

- 🍎 Apple-inspired UI/UX
- 🌙 Premium dark theme with red accents
- 📱 Fully responsive design
- 🎥 Smooth scrolling animations
- ✨ Scroll reveal effects
- 📊 Animated statistics counters
- 🖼️ Infinite services carousel
- ❓ Interactive FAQ accordion
- ⭐ Customer testimonials section
- 📍 Google Maps integration
- 📞 Quick contact buttons
- 💎 Glassmorphism navigation bar
- ⚡ Pure HTML, CSS & JavaScript (No frameworks)
